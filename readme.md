# Scala TDD for Math

Implementing math classes with Scala by TDD approach

  - Scala
  - TDD First
  - Math (Linear Algebra, calculus...)
  - Preparing for Machine learning, Deep learning and A.I.

### Installation and Running the project
Have SBT installed
```sh
$ cd project
$ sbt compile
$ sbt run
```
### Testing
```sh
$ cd project
$ sbt test
```
